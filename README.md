# 💪 Workout App

운동 습관을 게임처럼 즐기는 모바일 피트니스 앱.
캐릭터를 선택하고, 걸음 수를 쌓고, 포인트를 모아 보상을 받아요.

## 기술 스택 (모바일)

| 분류       | 사용 기술                     |
| ---------- | ----------------------------- |
| 프레임워크 | React 18 + TypeScript         |
| 빌드       | Vite                          |
| 스타일     | Tailwind CSS v3               |
| 라우팅     | React Router DOM v6           |
| 아이콘     | React Icons                   |
| 다국어     | react-i18next (한국어 / 영어) |
| 백엔드     | Supabase (예정)               |

## 실행 방법

```bash
npm run dev
```

## 화면 구성

### 🏠 홈

- 연속 운동 스트릭 뱃지
- 캐릭터 이미지 + 말풍선 메시지
- 걸음 수 / 이번 주 운동 / 포인트 저금 스탯 카드
- 운동 시작 버튼 → 트래킹 화면 이동

### 🧙 캐릭터

- 4개 캐릭터 유형 선택
- 캐릭터별 운동 스타일 · 포인트 보너스

| 캐릭터      | 특징             | 칼로리     | 보너스               |
| ----------- | ---------------- | ---------- | -------------------- |
| 🚶 산책러   | 천천히 걷기 특화 | 분당 3kcal | 연속 운동 포인트 2배 |
| 🚶‍♂️ 파워워커 | 빠르게 걷기 특화 | 분당 5kcal | 목표 달성 포인트 2배 |
| 🏃 러너     | 러닝 특화        | 분당 8kcal | 거리 달성 보너스     |
| 🏔️ 등산가   | 등산 특화        | 분당 6kcal | 파티 참여 보너스     |

### 🎉 파티

- 전체 파티 탐색 / 내 파티 탭
- 파티 만들기
- 🏆 주간 걸음 수 랭킹

### 💰 포인트

- 적립내역 / 선물하기 / 교환하기
- 친구에게 포인트 선물
- 카페·브랜드 기프티콘 교환 (준비중)

### 🥗 식단

- 칼로리 진행 바
- 캐릭터 유형별 맞춤 식단 추천
- 운동 후 단백질 섭취 알림

### 🏃 운동 트래킹

- 원형 링 게이지 + 캐릭터 애니메이션
- 실시간 거리 / 걸음 수 / 시간 / 칼로리
- GPS 기반 거리 측정 (예정)

### ⚙️ 설정

- 테마 선택: 에너지 / 자연 / 코스모
- 언어 전환: 한국어 / 영어
